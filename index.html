🧐 Sobre mim

Ramon Castro Barbosa

Desenvolvedor Full-stack com foco em Engenharia de Software e mais de 2 anos de experiência. Minha base de backend une a robustez do Java (Spring Boot) e a velocidade de desenvolvimento do TypeScript (NestJS), juntamente com à dinamicidade e inteligência do Python (FastAPI) para Análise de Dados com Machine Learning, tudo sustentado por SQL.

No Frontend, crio interfaces dinâmicas com React e NextJS, seguindo sempre os padrões de acessibilidade e desenvolvimento. Além do código, gerencio o ciclo de infraestrutura com Docker, Kubernetes e Terraform (IaC), integrando segurança e automação.
